\doxysubsection{Detailed Description}
This header file defines prototypes for all functions that are required by the state machine implementation.

This state machine makes use of operations declared in the state machine\textquotesingle{}s interface or internal scopes. Thus, the function prototypes\+:
\begin{DoxyItemize}
\item statechart\+\_\+display
\item statechart\+\_\+set\+Leds are defined here.
\end{DoxyItemize}

These functions will be called during a \textquotesingle{}run to completion step\textquotesingle{} (run\+Cycle) of the statechart. There are some constraints that have to be considered for the implementation of these functions\+:
\begin{DoxyItemize}
\item never call the statechart API functions from within these functions.
\item make sure that the execution time is as short as possible. 
\end{DoxyItemize}
